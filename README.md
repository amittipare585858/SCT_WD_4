# SCT_WD_4
Simple To-Do Web Application

Task 04: Multi-Tab To-Do App for Organizing Tasks & Lists

I have created a To-Do web application featuring multiple tabs for easy navigation and organization. Users can create different lists within the app to manage a variety of tasks. The app enables users to add, edit, and mark tasks as completed, while also allowing the setting of date and time for each item. This structured approach helps streamline productivity and keeps all personal or work-related tasks neatly organized.
